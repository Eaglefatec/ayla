# Ayla 

## 🎯 Desafio
O desafio consiste em otimizar e automatizar o processo de triagem manual de currículos em formato PDF realizado por equipes de Recursos Humanos.

Filtros tradicionais baseados em palavras-chave exatas geram gargalos operacionais e eliminam candidatos qualificados devido a preciosismo de termos, ignorando sinônimos e o contexto real da experiência profissional.

## 🏅 Solução
A **Ayla** é uma solução inteligente para triagem semântica e síntese de currículos baseada na arquitetura **RAG (Retrieval-Augmented Generation)**. 

A aplicação extrai o texto bruto de PDFs, converte as competências em vetores de contexto por meio do Ollama (IA generativa local) e realiza buscas sofisticadas baseadas em intenção.

Além de classificar os perfis mais adequados às vagas em linguagem natural, a Ayla gera *insights* automáticos e resumos executivos direcionados para apoiar a tomada de decisão do recrutador.

## 📋 User Story para o Produto

Como gerente de recrutamento... Quero utilizar uma aplicação em Java que analise currículos, usando modelos de linguagem (LLMs), para obter dados consistentes e transparentes dos candidatos, reduzindo o tempo e o viés humano na análise e seleção de Curriculuns no processo seletivo externo.
